# Besoin fonctionnel

## Problème

La génération de structures WinDev à partir de JSON est aujourd’hui manuelle,
chronophage et source d’erreurs.

## Objectifs

- Automatiser la génération
- Respecter strictement les règles de nommage
- Réduire les erreurs humaines

## Contraintes

- Sortie WinDev uniquement
- Normes internes obligatoires
