# JSON → WinDev Structure Generator

## Objectif

Développer un outil en Python permettant de transformer un JSON en structures WinDev
conformes aux normes internes Mecasystems / Emil Frey.

## Principe

- Analyse d’un JSON
- Inférence d’un schéma
- Génération de structures WinDev (ST*) via des règles YAML

## Cible

Développeurs WinDev / WebDev.
